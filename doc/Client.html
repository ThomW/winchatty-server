Access Methods

JSON

The JSON API is compatible with the classic chatty protocol. It can be used as the server URL in the iPhone application. Unlike the classic API, XML is not supported. The following REST requests are supported:

/chatty/about
Redirects to SqueegyTBS's site, for the benefit of his iPhone app which makes this API call to display his about screen. There is a 1% chance this will redirect to goatse.
/chatty/index.json
ClassicService.getStory(0, 1)
/chatty/Story ID.json
ClassicService.getStory(Story ID, 1)
/chatty/Story ID.Page.json
ClassicService.getStory(Story ID, Page)
/chatty/thread/Thread ID.json
ClassicService.getThread(Thread ID)
/chatty/search.json?author=Author&parent_author=Parent Author&terms=Terms
ClassicService.search(Terms + Parent Author, Author)
/chatty/stories.json
ClassicService.getStories()
/chatty/stories/Story ID.json
ClassicService.getArticle(Story ID)
/chatty/messages.json
ClassicService.getMessages(HTTP Username, HTTP Password)

For the new services which have no REST equivalent, you can use the JSON gateway. Send a GET request to:
http://yourdomain.com/service/json.php/Class.function/arg1/arg2/...

AMF

The AMF service gateway address is http://yourdomain.com/service/gateway. Flex/Flash can transparently call these APIs using the RemoteObject class.
